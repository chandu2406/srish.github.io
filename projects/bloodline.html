Bloodline : Facebook Developer World Hack Day

BloodLine is an android based mobile web social awareness application which connects blood seeking donors / recipients via facebook profile to donate or receive blood in urgency. This product is an outcome of 5 hours hack at Facebook World Hack Day 2012 at Bangalore.

Technical Implementations

Qt/ QML, Necessitas, Facebook Graph API
