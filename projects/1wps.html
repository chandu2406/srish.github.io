1WPS (One Wikipedia Per School)

1WPS is an acronym for 1 wikipedia per school. This project aims at providing offline e-learning content for rural based schools. It envisions that every school of a country should be equipped with offline e-learning content kiosk. I worked on the implementation of Wikipedia dump for this project and made use of open source formats ZIM (openzim.org, .zim file format is an open file format that stores wiki content for offline usage) and Kiwix (a GUI based zim reader). We created customized ubuntu version for dump integration using distribution remixing.
